<h1> REV NEO Brushless Motors V1.1 Overview </h1>

<h2> What is the NEO Motor? </h2>

The REV NEO is a brushless DC motor designed specifically for FRC robotics. It provides high power output in a compact package and integrates seamlessly with SPARK MAX motor controllers.

<h2> NEO Specifications </h2>

Key specifications:
- Free speed: 5,676 RPM
- Stall torque: 2.6 N⋅m (23 lb⋅in)
- Stall current: 105A
- Free current: 1.8A
- Brushless design for longer lifespan
- Built-in hall effect sensors

<h2> Common Uses </h2>

NEO motors are commonly used for:
- Swerve drive modules (drive and steering)
- Shooter wheels
- Intake mechanisms
- Elevator systems
- Any application requiring efficient, powerful motors
